Before clicking Submit, make sure you have configured your system to use `127.0.0.1` as a DNS server.

After clicking Submit, you will be forwarded to the spoofed host. Depending on yout browser, it may still point to the orginial site, in which case you'll need to flush the DNS cache.

I found Safari was the best browser, because you can simply call `sudo killall -HUP mDNSResponder`. Be warned! Chrome has MASSIVE problems when it comes to clearing the DNS cache. I gave up with Chrome.
